pymihome
===============================

version number: 0.0.1
author: Neil Griffin

Overview
--------

Library to access Energenie MiHome devices via the web API.

Work in progress.  Tune in later.

Installation / Usage
--------------------

To install use pip:

    $ pip install pymihome


Or clone the repo:

    $ git clone https://github.com/ng110/pymihome.git
    $ python setup.py install
